``fbpca`` - Functions for principal component analysis (PCA)
============================================================

Requirements
------------

- `numpy>=1.9`
- `scipy>=0.14`

Installation
------------

::

    pip install fbpca

Documentation
-------------

http://fbpca.readthedocs.org/
